Topic: Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It forms the structure of web content using a series of elements and tags.

Basic Structure of an HTML Document

  • Doctype: Declares the HTML version.
  • HTML Tag: The root element of an HTML page.
  • Head: Contains meta information, title, and links to stylesheets or scripts.
  • Body: Contains all visible content like text, images, and links.
